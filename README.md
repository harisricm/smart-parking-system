## Project Overview

The Smart Parking System is an IoT-based solution that detects parking slot availability and helps manage vehicle parking efficiently.

## Project Scope

This project demonstrates parking slot monitoring using sensors and ESP32 technology. It can indicate whether a parking space is occupied or available.

## Objectives

* Detect vehicle presence
* Monitor parking slot status
* Learn sensor integration
* Understand IoT-based automation

## Technologies Used

* ESP32

## Components Required
